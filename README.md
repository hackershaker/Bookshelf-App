# BookShelf App

This is BookShelf Project App. Started by [Jetpack Compose codelab][codelab].
Users can search books at Google Books, return list, and see books detail.

## Using Technique

- compose Navigation
- View model
- stateholder
- asynchronous HTTP request and response using Retrofit
